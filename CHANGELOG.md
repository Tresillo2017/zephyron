# Changelog

All notable changes to Zephyron will be documented in this file.

This project uses [Changesets](https://github.com/changesets/changesets) for versioning.

## 0.1.0 (2026-03-20)

### Features

- **Audio Streaming** -- Stream DJ sets from R2 with waveform visualization and live listener counts
- **AI Tracklist Detection** -- Automatic track identification from YouTube metadata, parsed by LLM, enriched via Last.fm
- **Community Annotations** -- Vote on, correct, and improve AI-detected tracklists
- **Self-Improving ML** -- Community feedback feeds back into detection prompts
- **Artist Pages** -- Enriched profiles with bios, tags, similar artists
- **Playlists & History** -- Personal playlists and listening history
- **Admin Dashboard** -- Set upload, invite code management, moderation, ML pipeline controls
- **Account Settings** -- Profile editing, password change, two-factor authentication (TOTP)
- **About, Privacy, Terms pages** -- Public informational pages

### Infrastructure

- **Versioning** -- Changesets for changelog and version management
- **CI/CD** -- GitHub Actions for lint + type-check on PRs
- **Dependabot** -- Automated dependency update PRs
- **GitHub Templates** -- PR template, bug report, feature request
- **MIT License**
- **Documentation** -- Complete README with setup instructions and project structure
